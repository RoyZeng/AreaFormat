# 地址信息格式化工具类库


---

地址信息格式化工具类库（**AreaFormat**）,主要用于对一些格式混乱的地址字符进行**识别，补全，格式化**
> * 识别不规则的地址信息（手动填写的地址信息）
> * 补全地址信息。如：“**湖北省 武昌区**” 转化 “**中国 湖北省 武汉市 武昌区**”
> * 地址信息格式化。如：“**钟祥**” 转化为 “**钟祥市**”，“**湖北**”转化为“**湖北省**” 
> * 类库提供多个方法调用 **（Standard）** 重装方法
